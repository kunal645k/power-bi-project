# Clothes E-Commerce Sales Dashboard | Power BI
## Project Overview
This Power BI dashboard analyzes clothing e-commerce sales data to uncover meaningful business insights related to sales performance, customer behavior, product categories, and profitability. The dashboard is designed to help stakeholders make data-driven decisions by visualizing key metrics in an interactive and easy-to-understand format.

## Key Objectives
- Track overall sales and revenue performance
- Analyze top-selling clothing categories and products
- Understand customer purchasing patterns
- Monitor profit, discounts, and order trends
- Identify seasonal and regional sales insights

## Dashboard Highlights
- Total Sales, Profit & Quantity Sold
- Category-wise and Product-wise Sales Analysis
- Monthly / Yearly Sales Trends
- Customer & Order Analysis
- Interactive slicers for dynamic filtering
